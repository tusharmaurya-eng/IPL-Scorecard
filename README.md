# Cricket TV ScreenCloud Ready

Run locally:
npm install
npm start

Open:
http://localhost:3000

Deploy online:
- Push this folder to GitHub
- Deploy the GitHub repo on Railway / Render / any Node hosting
- Add the public URL to ScreenCloud Website App

Included:
- Generic automatic current-scorecard URL generation
- Fallback to scorecard URL
- Vertical 4K scaling
- First page rotates after 10 seconds
- Other pages rotate after 25 seconds
- Auto-refresh every 3 minutes
